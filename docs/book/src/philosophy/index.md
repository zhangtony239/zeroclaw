# Philosophy

ZeroClaw is built on four opinions, in priority order.

- [You own it](./you-own-it.md) is the foundational constraint.
- [Security-first, with escape hatches](./security-first.md) follows from local-first.
- [Minimal](./minimal.md) keeps binary size, dependencies, and surface area small.
- [Provider-agnostic](./provider-agnostic.md) keeps the model pluggable.

See also [What this isn't](./what-this-isnt.md) and [How decisions get made](./how-decisions-get-made.md).
