Constants§

MEMORY_TOOL_NAMES
Canonical names of the long-term-memory tools. This is the single source of truth for “which tools touch the persistent memory store” — surfaces that need to strip memory access (e.g. ACP/Code sessions) consult this rather than re-listing tool names. Keep in sync with the Tool::name() each memory tool returns; the guard test memory_tool_names_match_tools fails if a memory tool is added or renamed without updating this list.
