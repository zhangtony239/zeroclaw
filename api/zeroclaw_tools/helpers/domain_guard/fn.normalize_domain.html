Function normalize_domain 

pub fn normalize_domain(raw: &str) -> Option<String>
Expand description

Normalize a single domain or allowlist entry to its canonical form.

Returns None for invalid entries (empty, whitespace, userinfo, unmatched IPv6 brackets).

§Bracket rules (maintainer requirement)

IPv6 brackets are only stripped when both [ and ] are present. Unmatched brackets (e.g. [::1, ::1], [127.0.0.1, 127.0.0.1]) are rejected.
