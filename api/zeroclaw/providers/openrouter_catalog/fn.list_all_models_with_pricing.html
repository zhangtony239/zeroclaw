pub async fn list_all_models_with_pricing() -> Result<Vec<ModelInfo>, Error>
Expand description

Return every OpenRouter model with pricing, keeping the full <vendor>/<slug> id. Sorted and deduplicated by id. Backs the first-class OpenRouterModelProvider::list_models_with_pricing so the cost-rates editor can prefill rates from the public catalog.
