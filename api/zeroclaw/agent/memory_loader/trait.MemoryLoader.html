Loads a memory-context preamble for a user message.

Implementations MUST emit a ObserverEvent::MemoryRecall event via observer for every recall call they perform — both on success and failure paths — so OTel/log observers can attribute per-turn memory cost. The agent runtime relies on this for end-to-end visibility of the implicit recall that runs at the start of each turn.
