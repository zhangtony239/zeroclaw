Build a router fronted by primary_name plus one provider per unique model_routes entry. Each dotted <family>.<alias> name resolves through the typed [providers.models.<family>.<alias>] config (endpoint URI, Azure resource, Gemini OAuth, etc.); bare family names use family defaults.
