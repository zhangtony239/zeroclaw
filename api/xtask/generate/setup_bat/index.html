Module setup_bat 

Expand description

setup.bat renderer. The hand-written imperative glue (toolchain bootstrap, PATH, copy, quickstart, prebuilt download) stays in setup.bat; only the drift-prone data lives in sentinel-delimited regions this renderer owns. setup.bat has two such zones - the build-mode menu/routing and the per-mode preset definitions - separated by the hand-written prebuilt block. Every label and feature string derives from the canonical Selection set.
