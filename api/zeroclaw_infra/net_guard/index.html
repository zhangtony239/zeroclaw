Module net_guard 

Expand description

Network-safety primitives shared across crates that must reject SSRF and local/private targets. Lives in zeroclaw-infra so both the tool layer (zeroclaw-tools domain guard) and the plugin host (zeroclaw-plugins zc_http_request) read one implementation without a tool-to-plugin dependency edge.
