Expand description

An RFC #7141 authentication provider: verifies one credential kind and emits a uniform AuthOutcome. Implementations live beside their identity source (e.g. oidc next to the IdP introspection code, native over PairingGuard).

Fail-closed contract: verify returns AuthOutcome::Denied for anything it cannot positively authenticate — never a silent allow.
