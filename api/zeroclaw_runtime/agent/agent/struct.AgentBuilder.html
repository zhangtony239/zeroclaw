pub fn exclude_memory(self, exclude: bool) -> Self

Exclude persistent memory from this agent. When set, the memory backend is replaced with NoneMemory, auto-save is forced off, and all memory_* tools are stripped from the tool set. Used by ACP sessions, which rely on session history for context rather than the agent’s long-term memory.
