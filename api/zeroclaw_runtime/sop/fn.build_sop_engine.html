Build a single shared SopEngine + SopAuditLogger pair.

This is the sole construction site for SOP state within a daemon. Callers receive Arc<Mutex<SopEngine>> and Arc<SopAuditLogger> handles — never call SopEngine::new or SopAuditLogger::new directly outside this module.
