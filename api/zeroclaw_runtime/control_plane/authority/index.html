zeroclaw_runtime::control_plane

Module authority 

Expand description

Runtime-authority guard — decides whether THIS process may reclaim a task.

Mirrors OpenClaw’s isRuntimeAuthoritative (task-registry.maintenance.ts): a daemon must never move another live daemon’s task to a terminal-loss state. Reclamation is safe only when the record is a prior-boot orphan, or its owning process is dead.
