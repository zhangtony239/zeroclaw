Module keymap 

Expand description

Keymap abstraction for zerocode.

Each leaf action enum carries its own default bindings inline. Consumers call ChatTabAction::from_chord(&key) directly — no Keymap struct, no plumbed argument.

On darwin, Chord::matches translates the CTRL modifier to SUPER so Linux’s Ctrl+K and macOS’s ⌘K resolve identically.
