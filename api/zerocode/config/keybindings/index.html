Module keybindings 

Expand description

Local keybinding presets and override resolution.

A preset is a named, COMPLETE keymap built from the typed action enums and Chord constructors — never authored "tag.variant" or "ctrl+p" strings. Picking a preset fully overwrites the [keybindings] table, so every preset must define every rebindable action; a test enforces it. Presets start from the full default map and reassign the actions they change. Walked from KEY_PRESETS, mirroring the theme registry walked via theme_names.
