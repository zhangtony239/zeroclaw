a2a

[a2a] section wrapper. Kept as a dedicated wrapper so the a2a table can host sibling sub-tables (outbound client config, signing) in later slices without moving server.

Server-level exposure gate for A2A agent discovery. Default-closed: the server serves nothing until [a2a.server] enabled = true, and even then only aliases that opt in via [agents.<alias>.a2a] published = true appear. The discovery routes serve on the gateway’s own listener; public exposure follows the gateway’s bind posture. See crates/zeroclaw-gateway/src/a2a.rs.

键类型默认描述
server对象—[a2a.server] — inbound A2A discovery server.
